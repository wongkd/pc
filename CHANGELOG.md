# 更新日志

## v2.5.0 (2026-06-06)

### 🖨️ 打印功能全面升级

- **横竖版区分**：打印 HTML 加入 `@page { size: A4 portrait/landscape }`，横版竖版各行其道
- **排版紧凑化**：字号缩小（表头 9px、正文 11px、标题 22px）、间距压缩，内容尽量一张 A4 打印完
- **空白过滤**：打印机箱、风扇等未填写项目不再显示空行
- **打印等待**：图片加载完再触发打印，避免 LOGO 空白
- **跨页优化**：`thead { display: table-header-group }` 翻页自动重复表头，`page-break-inside: avoid` 防止行断裂

### 🎨 版面与文字优化

- **移除"未填写"**：所有空字段留空，不再显示占位文字
- **LOGO 完整显示**：最大高度从 48px 放开到 64px，取消裁切
- **标语间距**：LOGO 与标语间距 10px，不再黏在一起
- **条款四项独立编辑**：付款方式、售后说明、质保政策、备注条款各一个输入框，右侧预览一一对应
- **联系人/电话**：留空时不显示无意义的斜杠分隔符

### ⚡ 加载性能优化

- **编辑器不再阻塞**：移除 `cloudLoading` 门禁，页面直接渲染本地数据
- **Logo 排除云端同步**：状态同步从 400KB 降到几 KB
- **骨架屏加载**：HTML 内联旋转 spinner，消除下载 JS 时的全白屏

### 🐛 其他修复

- **硬件库查价同步图片**：新增硬件时查价按钮自动加载产品图片
- **规格框默认清空**：从硬件库联想选择后规格保持空白
- **版本标签**：`v2.5.0`
